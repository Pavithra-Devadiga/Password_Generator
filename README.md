# 🔐 Password Generator

A responsive **Password Generator Web Application** that allows users to create secure random passwords based on selected criteria. The application also evaluates password strength and allows users to copy the generated password instantly.

🌐 **Live Demo:**  
https://pavithra-devadiga.github.io/Password_Generator/

---

## 📌 Overview

This project generates strong and random passwords using JavaScript.  
Users can customize password length and choose different character types such as uppercase letters, lowercase letters, numbers, and symbols.

The application runs completely in the browser and does not store any user data.

---

## ✨ Features

- 🔑 Generate secure random passwords
- 📏 Adjustable password length (6 – 24 characters)
- 🔤 Include:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
- 📊 Real-time password strength meter
- 📋 Copy password to clipboard
- ⚡ Instant password generation when the page loads
- 📱 Responsive and clean UI

---

## 🛠️ Technologies Used

- **CSS** – Styling and layout
- **JavaScript ** – Password generation logic
